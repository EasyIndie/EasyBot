🚀 快速开始

# 初始化配置目录
easybot --init

# 编辑 ~/.easybot/.env 填入凭据
# TELEGRAM_BOT_TOKEN=...
# DISCORD_BOT_TOKEN=...

# 启动服务
easybot --debug

支持配置文件热重载，修改 gateway.yaml 后自动生效
